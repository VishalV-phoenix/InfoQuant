# InfoQuant

## Purpose

InfoQuant is a multi-agent research and learning assistant that helps users understand technical topics through research, simplified explanations, and quiz generation.

## Architecture

- Orchestrator Agent
- Research Agent
- Tutor Agent

## Features

- Topic research
- Beginner-friendly explanations
- Quiz generation
- Session memory
- Tool calling via MCP
- Security guardrails
- Agent evaluation

## Constraints

- Maximum 3 agents
- CLI-based interface
- No authentication
- No vector database
- No external database
- Keep implementation simple
- Use Google ADK
